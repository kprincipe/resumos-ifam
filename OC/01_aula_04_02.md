# Evolução dos Computadores

## Elementos do Sistema Computacional
- O computador é uma máquina capaz de _executar uma sequência_ de instruções definidas pelo homem para gerar um determinado resultado.
- Hardware é a parte física desse sistema.

### Elementos de Hardware
- CPU, Memória Principal (RAM) e Dispositivos (periféricos).
- _Memória principal_ é a memória usada diretamente pelo processador, RAM, CACHES L1, L2 etc.. não são persistentes.

### PCs analógicos x Digitais
- *Computadores analógicos* não trabalham com números e são normalmente criados para finalidade específica.
    * Ex: Osciloscópio.
- *Computadores digitais* resolvem problemas realizando operações diretamente com números.
    * Ex: Notebook.

### Evolução Tecnológica
- Houve uma grande evolução desde o surgimento do computador analógico até o desenvolvimento do computador digital.
- O que impactou nessa evolução foram as tecnologias utilizadas nesses computadores.

## Gerações

### Geração Zero
- Caracterizada pelos computadores essencialmente analógicos, os quais eram construídos a partir de engrenagens mecânicas e eletromecânicas.
    * _Máquina de Pascal_: realizava contas de adição e subtração.

### Primeira Geração (1945 - 1955)
- Segunda Guerra Mundial foi marco do surgimento da computação moderna. Foi nesse contexto onde começaram a surgir novas tecnologias capazes de substituir os componentes mecânicos utilizado nos computadores analógicos.
- Exemplos desses componentes: relés, capacitores e as válvulas.
- As válvulas possibilitaram mais velocidade em comparação com os seus antecessores.

---

- Quebravam após algum tempo de uso;
- Exemplos:
    * ENIAC
        - Levou 3 anos para ser construído;
        - Funcionava aprox. 19.000 válvulas;
        - Consumia 200 kW;
        - Pesava 30 toneladas;
        - Tinhha 5,5m x 25m.

### Segunda Geração (1955 - 1965)
- Válvula substituída pelos _transistores_;
    - Reduziu significativamente o volume dos computadores;
    - Aquecimento mínimo;
    - Baixo consumo de energia;

### Terceira Geração (1965 - 1970)
- Surgem circuitos integrados: incorporam inumeros transistores e outros componentes eletronicos em _formato de miniaturas_ em um único encapsulamento.
- Cada chip e equivalente a inúmetos transistores.
- Criação dos primeiros sistemas operacionais.
- Linguagens COBOL, FORTRAN, Pascal, C, Basic.

### Quarta Geração (1970 - Atual)
- Evolução, principalmente na miniaturização dos componentes;
- Circuitos passaram a ter uma larga escala de integração - Large Scale Integration;
- Microprocessadores;
- Surgimento dos Computadores Pessoais;
- Javascript, Kotlin e outras linguagens bleh

### Quinta Geração (Presente - Futuro)
- Inteligência Artificial;
- Computadores Quânticos;
- Nanotecnologia;
- Multiprocessadores;
- Linguagem de alto nível;
- Linguagem natural;
- Processamento Paralelo;
- Portáteis;
- Leves;
- Maior velocidade;
- Maior memória;
- Reconhecimento de Linguagem Natural;
- Reconhecimento facial e de voz.

## Modelo de Von Neumann
- Contribuiu para a construção de um _computador sequencial binário_ de programa armazenado;
- Propos _elementos críticos_ de um sistema computacional.
